TA05 Portfolio Website
This repository contains a static portfolio website developed as part of the TA05 assignment. The objective of the project is to present a structured overview of my academic background, technical competencies, and selected projects related to systems administration, virtualization, and network services. The site is deployed using GitHub Pages.
