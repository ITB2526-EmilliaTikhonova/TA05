Intrusion Detection with Snort & Wazuh

Real-time threat detection and event correlation in enterprise networks

I deployed a network-based IDS using Snort and integrated it with Wazuh to monitor and analyze security events. The system detects suspicious activity and provides actionable insights through dashboards and alerts.

  • Snort deployment with custom detection rules
  • Wazuh integration for log analysis and alerting
  • SIEM dashboard setup with ELK Stack
  • Real-time monitoring of brute-force and scan attempts
