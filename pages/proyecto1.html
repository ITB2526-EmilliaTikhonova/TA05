Linux Server Hardening

Securing Ubuntu 22.04 for enterprise-grade resilience

This project focused on strengthening the security posture of a Linux server by applying hardening techniques. The goal was to minimize attack surfaces and enforce best practices for system integrity and access control.

  • SSH hardening (custom port, key-based auth)
  • Service minimization and firewall configuration
  • Password policy enforcement via PAM modules
  • Security auditing using Lynis
